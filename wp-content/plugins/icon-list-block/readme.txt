== Description ==

Create List with icon by this block plugin. If you want to make a list with icons this plugin is for you. It come up with many options to fulfill your needs.

- Install the Icon List Block plugin
- Add the Icon List block from the block category called "Widgets" in Gutenberg editor.
- You can change block settings from the right side settings sidebar.
- Enjoy!

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **Icon List Block**
3. Click on the **Icon List** to add the block

= Download & Upload: =
1. Download the **Icon List- Block** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`icon-list-block.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **Icon List Block** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
